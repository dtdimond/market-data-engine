# Market Data Engine

A high-performance C++ market data processing engine for real-time analytics.

## Features (Planned)

- Real-time market data ingestion from multiple sources
- Streaming analytics (VWAP, microstructure signals, volatility)
- Multi-threaded architecture with lock-free queues
- Options-specific analytics (Greeks, vol surface)
- Time-series storage
- Python integration for ML models

## Building

```bash
mkdir build && cd build
cmake ..
make
./market_data_engine
```

## Development

Built on macOS, Linux-compatible via Docker.
